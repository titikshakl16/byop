# 🌍 Budget Friendly Travel Planner

A web-based **Travel Planning Application** built using **Python Flask** that helps users create affordable travel plans based on budget, destination, and travel dates.

---

## Project Overview

The Budget Friendly Travel Planner allows users to:

* Enter destination
* Select travel dates
* Define travel budget
* Generate a complete travel plan instantly

The system provides:

✅ Transportation suggestions
✅ Accommodation options
✅ Food recommendations
✅ Tourist places
✅ Fun activities
✅ Day-wise travel ideas

This project demonstrates **full-stack development using Flask + HTML + CSS + JavaScript**.

##  Tech Stack

**Frontend**

* HTML5
* CSS3
* JavaScript

**Backend**

* Python
* Flask Framework

**Architecture**

* REST API using Flask routes
* JSON data communication
* Modular backend structure

##  Installation & Setup

Activate:

Windows
```
venv\Scripts\activate
```
Install Dependencies

```
pip install flask
```
Run Application

```
python app.py
```

Open browser:

```
http://127.0.0.1:5000
```
